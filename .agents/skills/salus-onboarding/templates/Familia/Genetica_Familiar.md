# Genética Familiar

> **O que é este arquivo:** aqui só entram condições/predisposições genéticas que aparecem em **2 ou mais membros com vínculo biológico** (cruzamento). Os achados genéticos individuais de cada pessoa ou animal — como resultado de mapeamento genético (ex: Genera), predisposições raciais, etc. — ficam em `Perfis/[Nome]/Genetica.md`. Se este arquivo estiver vazio ou curto, isso é normal: significa que ainda não há sobreposição confirmada entre membros, não que seus dados sumiram.
